Como gerar o token (passo a passo)
  1. Clique em "Gerar token →" acima (abre o GitHub)
  2. Em Note, dê um nome (ex: "Laudo Colono")
  3. Em Expiration, escolha "No expiration" para uso permanente
  4. Em Select scopes, marque apenas repo
  5. Clique em Generate token
  6. Copie o token gerado (começa com ghp_) e cole no campo acima

⚠️ O token só é exibido uma vez pelo GitHub. Guarde-o em local seguro.
